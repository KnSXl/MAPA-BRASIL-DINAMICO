## Configuração do Projeto

### Instalação de Dependências
```bash
npm install
```

### Compilar e Recarregar Automaticamente para Desenvolvimento
```bash
npm run serve
```

### Compilar e Minificar para Produção
```bash
npm run build
```

### Verificar e Corrigir Arquivos
```bash
npm run lint
```
